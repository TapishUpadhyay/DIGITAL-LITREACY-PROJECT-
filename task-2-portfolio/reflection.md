## Reflection (150–200 words)

For this task, I set up accounts on three platforms: GitHub, LinkedIn, and Kaggle. Each serves a distinct purpose in my academic and professional journey.

**GitHub** is a version control and code hosting platform used by developers worldwide. I created a profile README repository (named after my username) where I introduced myself with my name, branch, year, and what I hope to learn during my B.Tech — specifically, I mentioned wanting to build a strong foundation in programming and web development.

**LinkedIn** is a professional networking platform used by employers, internship coordinators, and industry professionals. I filled in my Education section with my current degree (B.Tech), institution (VIT Bhopal University), branch, and expected graduation year. I plan to update it progressively as I gain skills, projects, and internship experience.

**Kaggle** is a data science and machine learning community with datasets, competitions, and free courses. Even as a first-year student, creating an account early gives me access to learning resources and helps me build a data science portfolio over time.

Over the next four years, I plan to use GitHub to host all my project code, LinkedIn to connect with seniors and apply for internships, and Kaggle to practice data analysis once I have the foundational skills.
